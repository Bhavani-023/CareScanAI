# Care Scan AI

AI-powered Prescription Scanner and Medicine Reminder System.

## Features

- Upload prescription image
- Capture prescription using camera
- OCR text extraction
- Medicine detection
- Automated reminder schedule
- Browser notifications
- Responsive UI
- Cloud deployment on Render

## Tech Stack

- Python
- Flask
- OCR.Space API
- HTML
- CSS
- JavaScript
- Render
